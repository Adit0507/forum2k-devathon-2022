### Setup and Installation

### Instructions to run the app in your local system

- Download the code as a zip file
- Extract the file

````````````````````
cd Application Code
````````````````````

````````````````````
npm install
````````````````````
````````````````````
npm @ant-design/icons@4.7.0 axios@0.25.0 react-chat-engine@1.11.22
````````````````````
````````````````````
npm start
````````````````````
